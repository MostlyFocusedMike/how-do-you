How do you...

A quick reference guide for doing basic tasks in different languages

Create an indexed array

JavaScript


             
// create a new empty array
let arr = [];
let arr = new Array();
// create an array values inside 
let arr = ["foo", "bar"];
let arr = new Array("foo", "bar");
// arrays can hold objects, 
// other arrays, booleans, and variables 
let foo = "bar",
  obj = {"age": 25},
  otherArr = ["a", "b"];
// spaces are allowed: 
arr = [foo, 
  {"name": "Helen"}, 
  obj, 
  ["hello", "there"], 
  otherArr];

Creating an indexed array can be done either using a new Array object, or using literal notaion with the brackets.

Show value of an index and index of a value

JavaScript


// give index, get value 
let arr = ["a", "b", "c"]
console.log(arr[2]); // "c"
//give value, get index 
console.log(arr.indexOf("b")); // "1"
console.log(arr.indexOf("f")); // "-1"

If you know the index, just put it in brackets after the array. If you know the value, use .indexOf() method and pass the value you are looking for as an argument. If the value does not exist in the array, it will return a value of "-1" as the index.

Show length of array

JavaScript


// give index, get value 
let arr = ["a", "b", "c"]
console.log(arr.length) // 3

The .length method will return the length of the array

Change a value at a known or unknown index

JavaScript


let arr = ["a", "b", "c", "d", "e", "f"];
// known index
arr[3] = "bar";
// unknown index
arr[arr.indexOf("b")] = "foo";
console.log(arr); 
// ["a", "foo", "c", "bar", "e", "f"];
// be careful of indexes that don't exist 
arr[8] = "hame"
console.log(arr); 
// ["a", "foo", "c", "bar", "e", "f", undefined, undefined, "hame"]

To change the value at an index, just use bracket notation and redeclare the vlaue. To change a value at an unknown index, use the arr.indexOf("value") inside the bracket notation of the array.
If you try to declare the value of an index that is out of range of the array, you won't get an error, the array will simply fill in all intermediate indexes with a value of "undefined". This is why, when adding a value to the end of an array, you should always use one of the methods below, like .push().

Add a value to the end of an array

JavaScript


let arr = [];
arr.push("foo");
arr.push("bar");
console.log(arr);
  // ["foo", "bar"]
let x = arr.push("ham")
// x is now the value 3

In JS, use the .push() method to "push" a value onto the end of an array. This affects the original array and returns the new length of the array

Add a value to the start of an array

JavaScript


             
let arr = [];
arr.unshift("foo");
arr.unshift("bar");
console.log(arr);
  // ["bar", "foo"]

Use the .unshift() method to put an item at the start of an array. This affects the original array and the return value is the new length of the array.

Remove a value at the end of an array

JavaScript


             
let arr = ["a", "b", "c"];
let popped = arr.pop();
console.log(arr);
// ["a", "b"]
console.log(popped);
// "c"

The pop() method removes the last element of an array, and returns that element. This affects the actual array itself.

JavaScript

// ["b", "c"]

The shift() method removes the first element of an array, and returns that element. This affects the actual array itself.

Join an array into a string

JavaScript

            
let arr = ["a", "b", "c"];
console.log(arr.toString());
// a,b,c
console.log(arr.join(", then "));
// "a, then b, then c

// join() can also take variables as arguments
let foo = " wow ";
console.log(arr.join(foo));
// "a wow b wow c"

foo = ["foo", "bar"];
console.log(arr.join(foo));
// "afoo,barbfoo,barc

To join the items in an array into a string there are two main ways: toString() and join().
toString() will join the array values into a string separated only by a comma.
join() takes an argument and then inserts that between the values of the array.
It's intersting to note that if the argument is an array or number it will be converted into a string using the toString() method.

Insert/remove items anywhere in an array

JavaScript

            
// including index 2, remove 1 item (c)
let arr = ["a", "b", "c", "d", "e", "f"];
arr.splice(2, 1);
console.log(arr);
// ["a", "b", "d", "e", "f"]
// you can remove as many items as you like
// just change the second argument

//starting at index 3, remove 0 items, put int "foo" and "bar"
arr = ["a", "b", "c", "d", "e"];
arr.splice(3, 0, "foo", "bar";
console.log(arr);
// ["a", "b", "c", "foo", "bar", "d", "e"]
// note: the new items will START on index 3

// splice() returns only the items that it removes 
console.log(arr.splice(2,2, "foo"));
// ["c", "d"]
// notice the "foo" is ignored 
console.log(arr);
// ["a", "b", "foo", "e", "f"]
// but see how it still affects the array

Using the splice() method works like so:
arr.splice(index, num ofitems to remove, items to add, another item, ...) So you can use the method without any items to remove things from an array, or you can add items into the array.
NOTE: the index is inclusive, so if you put 3, it will delete or add items in ON the third index.

Clone a slice of or an entire array

JavaScript

            

let arr = ["a", "b", "c", "d", "e", "f"];
// copy the entire array
let newArr = arr.slice(0);
console.log(newArr);
// ["a", "b", "c", "d", "e", "f"];

// copy from (including) the index 2
newArr = arr.slice(2);
console.log(newArr);
// ["c", "d", "e", "f"]
  
// copy from the third to last value 
newArr = arr.slice(-3);
console.log(newArr);
// ["d", "e", "f"]
 
// copy from the index 2 up to index 5
newArr = arr.slice(2, 5);
console.log(newArr);
// ["c", "d", "e"]

// copy from the index 2 up to the last index
newArr = arr.slice(2, -1);
console.log(newArr);
// ["c", "d", "e"]

// copy from the 4th to last up to the last 
newArr = arr.slice(-4, -1);
console.log(newArr);
// ["c", "d", "e"]

The .slice() method creates a new array without affecting the original. It takes two arguments, the first is the index to start the slice on, the second is where to go up to.
The first index is inclusive, the second is exclusive. If only one argument is given, it is inclusive. Negative numbers can be used, these will be indexes from the end (-1 is the last, -2 is second from the last). So arr.slice(0) is how to simply clone an array.

arr.length = 0;
// this changes the actual array, so that any and all 
// references to it now draw upon an empty array

//bad method

...code
arr = [];
// this creates a NEW blank array, but doesn't 
// erase the old one, so previous references mess up

While unintuitive, setting the length of an array to 0 will delete the contents in the actual array and take care of any reference issues.

Sort an array numerically

JavaScript

            
let arr = [10, 40, 2, 30, 50, 45, 71, 7]; 
  
// lowest to highest 
arr.sort(function(a, b){return a - b;});
  console.log(arr);  
  // [2, 7, 10, 30, 40, 45, 50, 71]


// highest to lowest 
arr.sort(function(a, b){return b - a;});
  console.log(arr);
  // [71, 50, 45, 40, 30, 10, 7, 2]

The .sort() method takes a sorting function that tells it to subtract two values from one another. This gets a positive number, negative number or 0, which tells .sort() the value of each number in the array compared to the other numbers.
a-b returns smallest to highest and b-a returns the opposite.

Sort an object array

JavaScript

            
// an array made up of objects, each with a 
// numeric property value (i.e. year)
var cars = [
    {type:"Volvo", year:2016},
    {type:"Saab", year:2001},
    {type:"BMW", year:2010}
  ];
// use the same sort function, just add dot notation
cars.sort(function(a, b){return a.year - b.year});

Sorting objects by their property value is a simple matter of using dot notation to access the property value that you want. The same sorting function applies.

Convert an array like object (html node list) into an array

JavaScript

            

// get your node list
var items = document.getElementsByTagName("li");

// this is the actual conversion line
var arr = Array.prototype.slice.call(items);


// use array only methods on the actual nodes
// as you can see, this will change the color of every node
arr.forEach(function(item, index){
  item.style.color = "blue";
});

// another example of an array only method
let thing = document.getElementById("foo");
console.log(arr.indexOf(thing));
// for instance, it will show an index of 1
// since that is where that specific node is

HTML node lists can have a length and be looped through, but that's it. If you want to use methods like .forEach, indexOf, .pop, then you will need to create an array.
Do this by using the Array.prototype's slice method, and use call to change the value of this to the array like object you want to convert. Now, using that new, real array, you can manipulate and use any array method you want.

Find the max and min of an array

JavaScript

            
let arr = [1, 4, 100, 12, 40];
// find the max value of an array
console.log(Math.max.apply(null, arr));

// find the min value of an array 
console.log(Math.min.apply(null, arr));

// find max value of a set of numbers 
console.log(Math.max(1,2,3,4));

// find the min value of a set of numbers 
console.log(Math.min(1,2,3,4));


// the above methods are the simplest,
// but below are the fastest way to find
// the min and max of an array 

// max
function myArrayMax(arr) {
    var len = arr.length
    var max = -Infinity;
    while (len--) {
        if (arr[len] > max) {
            max = arr[len];
        }
    }
    return max;
}

// min
function myArrayMin(arr) {
    var len = arr.length
    var min = Infinity;
    while (len--) {
        if (arr[len] < min) {
            min = arr[len];
        }
    }
    return min;
}

Use the Math.max or min to find the max and min, but if you are using an array and not just a set, you need to use the .apply method to change the value of "this" to be null and then take the array as the argument. Here is an excellent article explaining why you use apply and not call (call is for variables, apply is for arrays).
The two functions listed are the fastest way to do this, but more verbose.
